              • void addFunction(Map setting)

                機能のインスタンスを生成します。
                機能設定として以下のキー、値を格納したマップを渡してください。

                key
                機能を特定するキー。必須。
                class
                機能の実装クラス。必須。
                initial
                機能を初期化するためのクロージャ。省略可。詳細は各機能の実装クラスを参照してください。

                setting - 機能設定
