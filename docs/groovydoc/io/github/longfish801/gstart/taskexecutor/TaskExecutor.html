Package: io.github.longfish801.gstart.taskexecutor

[Groovy] Class TaskExecutor

      • io.github.longfish801.gstart.taskexecutor.TaskExecutor

      • NoticeTaskCallerを継承し、タスクを非同期実行します。
        タスクとは、実行状況を確認可能なタスクです。
        具体的にはNoticeTaskを継承したタスクです。
        startCheckThread()を呼ぶと、タスクの実行状況スレッドを開始します。
        タスク実行で例外が発生したならばクロージャを呼びます。
