必要な環境 インストール手順

必要な環境

　gstartは Windows OS上での利用を想定しています。

　実行には、以下の環境が必要です。

  • Java Runtime Environmentの 1.8.0_141以上が動作すること。

　ビルドには以下の環境が必要です。

  • Java Developers Kitの 1.8.0_141以上が動作すること。
  • Gradle 4.0以上が動作すること。
  • インターネットに接続できること。

インストール手順

　gstart.exeをダブルクリックして起動してください。
　タスクトレイアイコンに gstartのアイコンが表示されます。
